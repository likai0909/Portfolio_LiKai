Hotel Booking Website

Developed a hotel booking website that allows users to easily book stays at various hotels. The frontend was built using HTML, CSS, and EJS for dynamic content rendering, providing a responsive and user-friendly experience. On the backend, Node.js was used to handle server-side logic, manage bookings, and interact with a PHPMyAdmin database to store hotel and booking information. The project focuses on seamless user interaction and efficient data handling for a smooth booking process.

Skills Used

  • Node.js
  • CSS
  • PHPMyAdmin
  • JavaScript
